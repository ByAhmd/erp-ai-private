# Saudi Compliance Notes

ERP AI is designed as a Saudi-first ERP and accounting platform, but this initial skeleton is not a legal compliance certification.

ZATCA e-invoicing will be implemented in a later stage. This repository does not yet include real ZATCA XML, QR generation, cryptographic stamps, Fatoora integration, or clearance/reporting flows.

VAT rules require Saudi tax review before production use. Zakat and Withholding Tax also require Saudi compliance review before any filing, reporting, or calculation logic is sold to companies.

GOSI and WPS/Mudad support will be implemented later as part of the payroll phase.

Before a production release is sold to companies, the final system must be reviewed by a Saudi accountant, tax advisor, or compliance specialist.
